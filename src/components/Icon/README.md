## Icon(图标)

#### 属性

属性名|类型|默认值|说明|其他值
:---		|:--		|:--		|:--		|:--
name	|string	|		|图标名	|
color	|string	|#ddd		|图标颜色	|
className|string| |自定义class|
onClick	|function	|		|点击事件	|

#### 例子

###### name的值为下方的 icon-right 中的去掉icon-的名字

```
<Icon name="right" color="#ff6600" />
```
